# 2026-5_C++_Lab_6

## Overview

6. laboratorijas darbs

Šablonu bibliotēkas izmantošana


Pārveidot 4. laboratorijas darbā izstrādāto programmu, ievērojot zemāk minētās prasības.
* Objektu sarakstu 4. darbā izveidotajā klasē aprakstīt kā noskaņojamo sarakstu list<...> no STL bibliotēkas.
* Šajā klasē realizēt tikai vienu konstruktoru bez parametriem.
* Izņemt no šīs klases atribūtus, kas nosaka masīva izmēru (piemēram, DEFAULT_MAX_SIZE, MaxSize, Size) un metodi operator++(), kas paredzēta masīva elementu skaita palielināšanai.
* Pārveidot šīs klases metodi operator+=(), lai tā izmantotu STL klases list iespējas.
* Pārveidot šīs klases informācijas izvades metodi Print(), lai tā izmantotu iteratorus.
* Pārveidot destruktoru saraksta likvidēšanai (atmiņas atbrīvošanai).
* Elementu pievienošanas procesā saraksta garumu nekontrolēt un izņēmumus neradīt.
* Atrast prasāmo maksimālo vai minimālo elementu sarakstā, lietojot STL funkcijas. Atļauts izmantot lambda izteiksmes.
