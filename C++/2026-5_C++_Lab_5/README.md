# 2026-5_C++_Lab_5

## Overview

5. laboratorijas darbs

Šabloni


Pārveidot 4. laboratorijas darbā izstrādāto programmu.

Aprakstīt 4. laboratorijas darbā esošās klases (izņemot izņēmuma objektu klasi) kā šablonus ar vienu parametru T ( template <class T>). Šablona parametrs T jāizmanto kā tips vienam no klases atribūtiem, saskaņā ar darba variantu.
Izmantojot šo šablonu, izveidot divas klases objektu objektu saraksta glabāšanai ar dažādiem uzdevumā norādītā atribūta tipiem.
Izveidot abu klašu objektus, pārbaudīt dažu metožu darbu un izvadīt informāciju par katra objekta stāvokli.
