# 2026-4_C++_Lab_4

## Overview
4. laboratorijas darbs

Operāciju definēšana

Darba risinājums jāveido, izmantojot 3. laboratorijā darbā izveidoto programmu. Risinājumā jābūt definētam pievienošanas operatoram +=, kas pievieno sarakstam jaunu objektu, un operatoriem <<, kas realizē izvadi plūsmā.

Klasē, kas realizē objektu sarakstu, aizvietot metodi Add() ar definētu pievienošanas operatoru +=.
Visās klasēs aizvietot metodi Print() ar definētu plūsmas izvades operatoru <<, kas realizēts kā ārēja funkcija un ir šīs klases draugs (friend). Tas jāizdara trīs klasēs – virsklasē no 1. laboratorijas darba, apakšklasē no 2. laboratorijas darba un klasē-kompozīcijā  no 3. laboratorijas darba.
Funkcijā main() izmantot operāciju +=, lai pievienotu sarakstam jaunus elementus, kā pārbaudīt izvadīšanu plūsmā, izmantojot definēto operāciju << .
